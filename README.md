# Reactive Apps Without Frameworks

Talk deck basato su `@pixu-talks/theme` con i contenuti di `vanilla-signals-talk`.

## Include

- slide su Signals, DOM Parts, Proxy Store e template literal taggati
- demo Todo embedded nel deck
- runtime vanilla con `Signal.State`, `Signal.Computed`, `effect`, `model` e `repeat`

## Avvio

```bash
pnpm install
pnpm start
```
